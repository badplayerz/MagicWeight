终身保持期

好了,经过上面三个阶段,你已经脱胎换骨了。相信你再也不想回到过去那么肥胖的样子了。接下来就进入第四阶段,终生保持期。

经过上面三个阶段的训练,你现在的代谢系统已经得到修复,相对以前更不容易长胖。

代谢系统已经修复的特征是什么?非常容易启动脂肪代谢,对运动和减少热量摄入非常灵敏,不容易产生饥饿感。

现在你可以完全恢复正常的饮食,一日三餐基本可以随便吃,不要胡吃海喝即可,尽量避免吃夜宵。为了保持代谢系统长期的灵敏度,建议可以时常吃一下粗粮(第二阶段的食物)。

现代人难免应酬交际,胡吃海喝可能难以避免。往往一顿胡吃海喝可以让你的体重增加1-2斤。请注意不要连续多天胡吃海喝。在某天暴饮暴食之后几天略微减少一下饮食摄入,你的体重很快就会恢复正常。(靠良好代谢系统的自我调节能力)

你不再需要和减肥期间一样每天称体重,不过保持经常称体重的习惯还是有必要。

运动不是减肥的好方法,确是保持体重的好方法。现代研究表明,运动对保持胰岛素的灵敏度有很大的好处。

另外请反思,当年让你长胖的原因是什么?比如大量喝含糖饮料?经常喝很多啤酒?请将这些坏习惯改掉。

退一万步讲,你过了很多年后又长了几斤,很简单,执行第二阶段一段时间,就可将体重减回,千万不要再累计到几十斤再减。
