完整科学的减肥方法必须分四个阶段

长期肥胖者由于代谢系统钝化,身体长时间没有燃烧过脂肪,启动减肥非常困难。因此减肥必须设计第一阶段快速启动脂肪燃烧的方法。有些减肥方法虽然没有提到分阶段的概念,但在实际方法中有一些这样的意识,比如作者看到过一种吃水果减肥法,要求前三天完全绝食只喝水(太狠了)。

在启动减肥之后,还要设计第二阶段持续脂肪燃烧的方法。启动难,效果持续更难,一个肥胖者要减去几十斤,至少要几个月时间,要保证减肥的方法在几个月内都有效才行。

减去体重后,只是第一步,一个人在短期内减去几十斤,很容易反弹。必须要设计第三阶段防止反弹的方法。当年会长胖,必然有原因的,在减肥成功后,如何避免重蹈覆辙?这就是第四阶段终身保持期。

一个真正系统的减肥方法,必须包含这四个阶段。否则就不是彻底解决肥胖问题的完整减肥方法。
